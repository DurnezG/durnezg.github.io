Emerald

Emerald is a custom game engine I am developing in C++ to support my personal projects. These projects both being game related and electronics/embedded-related. Currently, it provides the core framework for rendering, input handling, and resource management. They still need a lot of work, but they are functional enough to start building upon. In the future, I plan to add more editor integration, scene management, and Vulkan-based rendering features to fully support complexer projects.

The code for this project is available on GitHub: Emerald
