Sapphire Editor

Sapphire is my in-development editor designed to work seamlessly with my game engine. Currently, it allows basic scene management, entity creation, and component editing. You can add objects, set up their transforms, and modify properties directly in the editor.

In the future, I plan to expand its functionality to include more advanced tools such as material editing, animation previews, and integrated scripting support. The goal is to have a robust environment where I can design and test scenes quickly before running them in the engine.
