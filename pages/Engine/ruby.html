Ruby

Ruby is a custom renderer I am developing to work alongside my Emerald engine. Currently it only sets up vulkan with a basic triangle rendering example, but it is functional and serves as a foundation for future features. I would like to add more support like in my previous GP2 render project but due to the lack of time I keep postponing continuing this project. In the future, I plan to implement deferred rendering, post-processing effects, and a PBR pipeline to fully integrate with my engine and editor workflow.
